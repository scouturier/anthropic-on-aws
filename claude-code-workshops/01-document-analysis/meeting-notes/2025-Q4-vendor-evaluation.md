# Loyalty Platform Vendor Evaluation
**Prepared by:** Marcus Williams, Head of Partnerships
**Date:** October 22, 2025
**Status:** FINAL — For Decision Review

## Background
Following the August product review and September customer research, the team agreed to evaluate external loyalty platform vendors to support the rewards restructuring. Two vendors were shortlisted from an initial RFP of six.

## Vendor A: LoyaltyCore (Incumbent)
**Overview:** Our current loyalty platform provider since 2019. Canadian-based, mid-market focus.

| Criteria | Rating | Notes |
|----------|--------|-------|
| Platform capability | 3/5 | Can support 3-tier model. Points system would require custom development (~4 months). |
| Integration effort | 4/5 | Already integrated with our core banking system. Migration risk is minimal. |
| Cost (3-year) | $2.1M | Includes platform upgrade + custom points module. 15% discount offered for early renewal. |
| Timeline to launch | 5 months | Includes 4 months dev + 1 month testing. |
| Partner API support | 3/5 | WestJet integration would need rework. Basic API documentation. |
| Data & analytics | 2/5 | Limited real-time reporting. Would need supplementary analytics tooling. |

**Key risk:** Their analytics capabilities are weak. We'd need to maintain a separate reporting layer, adding ongoing operational cost.

## Vendor B: RewardStack (New)
**Overview:** US-based, Series C startup. Strong in fintech loyalty. Used by Chime, Current, and several mid-tier US banks.

| Criteria | Rating | Notes |
|----------|--------|-------|
| Platform capability | 5/5 | Native support for tiers + points + hybrid models. Highly configurable. |
| Integration effort | 2/5 | Full new integration required. They provide SDKs but our core banking system isn't in their standard connector library. |
| Cost (3-year) | $2.8M | Higher base cost but includes analytics suite and partner management portal. |
| Timeline to launch | 8 months | Includes 5 months integration + 2 months testing + 1 month parallel run. |
| Partner API support | 5/5 | Robust partner API with self-service portal. WestJet integration would be straightforward. |
| Data & analytics | 5/5 | Built-in real-time dashboards, A/B testing for reward offers, predictive churn modeling. |

**Key risk:** Longer implementation timeline. US-based company — data residency needs to be confirmed for Canadian compliance.

## Recommendation

**I recommend Vendor B (RewardStack).**

While the cost is higher and timeline longer, the platform capabilities are significantly stronger. The built-in analytics alone would save us the cost of maintaining a separate reporting layer (~$400K/year). The partner API would future-proof our co-brand relationships. And the A/B testing capability means we can iterate on the rewards structure post-launch rather than betting everything on the initial design.

The 8-month timeline means a Q2 2026 launch is tight but achievable if we start procurement in November.
